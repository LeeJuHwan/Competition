### 결론 - 프로젝트 요약
사용자 중심의 공공데이터 포털을 만드는 것이며 의사결정을 도울 수 있는 생성 AI를 도입 하여 아이디어의 확장을 돕는 서비스를 목표로 하였습니다.

1. 브레인스토밍
2. 사용자 입력 데이터 외에 유사데이터 제공
3. 데이터 시각화
4. 사용자 친화적인 그래프 생성
